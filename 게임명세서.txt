#Rule
하나의 카드 덱에서 순서대로 카드를 뽑을지 말지(고스톱처럼) 결정함.
카드 숫자 총합이 딜러보다 21에 가까우면 승리 베팅금액 나눠가짐 -> 21이면 무조건 승리 베팅금액 1.5배 획득(100 베팅시 250 돌려받음)
총합이 21을 넘어가면 패배 -> 베팅금액 잃음

#Sequance
게임 시작 {
딜러(서버) 클라이언트1이 접속하고 y/n 수락 시 게임 시작 -> 서버 측에서 알림 전달함

n라운드 시작 {

클라이언트1이 카드를 한장씩 배분 받음

클라이언트1 베팅 금액 결정 후 STAY or HIT 결정 -> STAY한 경우 or 카드 총합이 21이 넘어간 경우 현재 본인의 점수 총합, 베팅 금액 서버에 전달

서버측에서 21을 초과한 클라이언트는 패배 처리

딜러측 STAY or HIT 진행 -> 각 클라이언트1의 카드 총합과 비교하여 각 클라이언트의 승패 여부, 베팅 금액을 다시 전달.
(플레이어 구조체에 베팅금액 속성을 추가해서 그냥 플레이어 구조체를 주고받을 생각 중)

각 클라이언트에서 서버에서 보낸 승패 결과, 베팅 금액 수령.

매치 종료.
}
}
